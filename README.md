## Installation

```bash
npm install snap-fade-away
# or use yarn
yarn add snap-fade-away
```

## Usage

```javascript
import snapFadeAway from 'snap-fade-away'

async (() => {
  await snapFadeAway(window.root)
  // Animated!
})()
```
